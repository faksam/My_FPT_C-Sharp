CF112 Course Outline

Course Code: CF112
Course Name: C# & .NET Technology
No of credits: 4
Course Description: This course helps students learn and practice
1. C# language,
2. .NET technology, including .NET platform, .NET framework, data access, ADO.NET, Windows Forms applications, Web Forms applications and Web Services.
Prerequisite: CF100, CF102
Time Allocation: Contact time: 39 slots
Lectures : 18
Lab/Tutorials: 18
Presentation : 3
1 slot = 90'
Main Objectives: Upon completion of this course students shoud:
1. Understand the followings: (ABET k1)
• C# language for developing .NET applications;
• Fundamental concepts of .NET 3.5 Platform
• Basic knowledge of WinForms .NET
• Basic knowledge of WebForms .NET
2. Be able to : (ABET k2)
• Develop Windows applications with Winforms .NET
• Develop Windows applications with WebForms .NET
3. Be able to apply DSA, DB knowledge and .NET technology to solve in solving problems (ABET a)
4. Be able to work in team and present group's results (ABET g)
Topic Outline:
  • Introducing C# and the .NET platform - 5%
  • The C# programming language - 20%
  • Windows Forms application - 25%
  • Database access with ADO.NET - 25%
  • Web Forms applications and Web Services- 25%
Main Textbook: Pro C# with .NET 3.0
Reference: 1. MSDN library
2. C# How to Program, Prentice Hall
Tools: 1. Microsoft Visual Studio 2008 Professional Edition
2. Microsoft SQL Server 2005 Express Edition
Promotion Policy: To obtain a credit in this subject, a student must:
  • Achieve a grade of 40% or better on the final exam
  • Achieve a grade of 50% or better on the overal course
Evaluation: 1. Ongoing assessment:
  • Workshops     : 15% (Abet k2)
  • Quizzes          : 15% (Abet k1)
  • Assignments   : 10% (Abet k2)
  • Presentation    : 10% (Abet k2)
  • Practical Exam: 20% (Abet k2)
2. Final Exam: 30% (Abet k1)
